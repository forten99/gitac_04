- Git 버전관리 프로세스
  - 준비단계
    - remote repository 생성
    - clone : 로컬 위치에 repository를 가져옴
  - 작업단계
    - commit : 수정 확인
    - push : remote repository에 업로드
  - 협업단계
    - branch
    - pull
